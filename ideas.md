# ESB DO Dashboard — Design Brainstorm

## Three Approaches

### Approach A: Regulatory Command Centre
**Theme:** Dark, data-dense, Bloomberg Terminal aesthetic
**Intro:** A dark-mode dashboard with high-density tables, monospaced typography, and sharp accent colours. Feels like a financial trading terminal — authoritative and serious.
**Probability:** 0.07

### Approach B: Civic Infrastructure Dashboard (CHOSEN)
**Theme:** Clean government/utility sector aesthetic — ESB corporate identity
**Intro:** A light, structured dashboard using ESB's official Navy (#003A70) and Green (#007A3D) palette. Combines the precision of a regulatory reporting tool with the accessibility of a modern SaaS product. Professional, trustworthy, and immediately legible to non-technical stakeholders.
**Probability:** 0.08

### Approach C: Bold Data Visualisation
**Theme:** Vibrant, chart-forward, Figma/Notion-inspired
**Intro:** A colourful, chart-heavy dashboard with large data visualisations, gradient accents, and a playful but professional feel. Prioritises visual impact over density.
**Probability:** 0.03

---

## Chosen Approach: B — Civic Infrastructure Dashboard

### Design Movement
Government/Utility sector corporate design. Think ESB Annual Report meets modern SaaS dashboard.

### Core Principles
1. **Regulatory precision:** Every number is traceable. No ambiguity in data presentation.
2. **Hierarchy through weight:** Playfair Display for headings, Inter for body. Bold numbers, quiet labels.
3. **ESB brand fidelity:** Navy (#003A70) as primary, Green (#007A3D) as success/positive, with standard RAG colours for risk.
4. **Density without clutter:** Tables are information-dense but use generous row height and clear column separation.

### Color Philosophy
- **Background:** #FAFAFA (off-white, not pure white — warmer and easier on the eye)
- **Primary (Navy):** #003A70 — headers, sidebar, primary actions
- **Success/Green:** #007A3D — positive RAG, on-track indicators
- **Amber:** #F59E0B — at-risk indicators
- **Red:** #DC2626 — critical/missed indicators
- **Blue:** #2563EB — informational/not-started indicators
- **Text:** #1A1A1A primary, #6B7280 secondary

### Layout Paradigm
Persistent left sidebar (240px) for navigation. Main content area with a top header bar showing the reporting period and last updated timestamp. Three main views accessed via sidebar links.

### Signature Elements
1. RAG status badges with consistent pill styling and subtle background tint
2. Penalty exposure numbers displayed in large, bold typography with euro sign
3. ESB Navy sidebar with white text and green active indicator

### Typography System
- **Headings:** Inter 700/800 (not Playfair — this is a data tool, not a presentation)
- **Body:** Inter 400/500
- **Numbers/Data:** Inter Mono or tabular numbers for alignment
- **Sizes:** Page title 24px, Section heading 18px, Table header 13px, Body 14px

### Brand Essence
ESB PR6 Delivery Obligation Scorecard — for PMO Senior Team Leaders who need to protect €18.9bn in regulatory commitments.

### Signature Brand Color
ESB Navy #003A70
